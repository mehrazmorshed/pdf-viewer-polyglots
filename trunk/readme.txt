Read PDF in multiple language.

== Description ==

This plugin allows you to read PDF in multiple language.

== Installation ==

Install the PDF Viewer Polyglots plugin via WordPress Dashboard (from plugin directory or from your computer), or upload the files manually to your server and activate the plugin.

= Install via WordPress dashboard =

*From Plugin Directory*

* Visit ‘Plugins > Add New’
* Search for ‘PDF Viewer Polyglots’
* Install and Activate the plugin

*From Your Computer*

* Download the PDF Viewer Polyglots plugin
* Visit ‘Plugins > Add New > Upload Plugin’
* Choose the plugin file in .zip format
* Click the Install Now button, and Activate after installation

= Install via Uploading Files Manually =

* Download the PDF Viewer Polyglots plugin and unzip the folder
* Upload the folder to your ‘/wp-content/plugins/’ directory
* Activate the PDF Viewer Polyglots Plugin from Dashboard

== Frequently Asked Questions ==

= How to read PDF in different languages? =

Activate the plugin and select a language from the settings page. That's it.
